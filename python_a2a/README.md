# Python A2A Exploration

This directory contains exploration and analysis of the Python A2A library.

## Structure

```
python_a2a/
├── A2A/                    # Python A2A virtual environment and docs
│   ├── docs/              # Python A2A documentation
│   ├── examples/          # Python A2A examples
│   └── ...
├── explore_a2a.py         # Exploration script
└── a2a_exploration.json  # Exploration results (if exists)
```

## A2A Directory

The `A2A/` directory contains:
- Documentation for Python A2A library
- Runnable examples
- Virtual environment (if exists)

## Exploration Script

Use `explore_a2a.py` to explore the Python A2A library structure:

```bash
cd python_a2a
python explore_a2a.py
```

## Related Directories

- `../adk/` - Google ADK documentation and examples
- `../a2a_sdk/` - A2A SDK (official) documentation and analysis
